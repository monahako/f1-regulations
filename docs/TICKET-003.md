# TICKET-003: 単年表示ロジック（app.js）

## 概要

単年表示画面（画面A）の動作ロジックを実装する。年度選択に連動して車体画像・レギュレーション内容を切り替え、アコーディオンの開閉を制御する。

## 対象ファイル

- `app.js`（新規作成）

## 依存チケット

- TICKET-002（index.html の DOM 構造が必要）

## タスク

### 画像マッピング
- [x] 年→画像ファイル名の変換関数 `getCarImage(year)` を実装する
  | 年範囲 | ファイル名 |
  |---|---|
  | 2000〜2008 | `CarImage_2000-2008.png` |
  | 2009〜2011 | `CarImage_2009-2011.png` |
  | 2012〜2021 | `CarImage_2012-2021.png` |
  | 2022〜2025 | `CarImage_2022-2025.png` |
  | 2026 | `CarImage_2026.png` |

### Markdownコンテンツ読み込み
- [x] `fetch('regulations/{year}.md')` で選択年の.mdファイルを取得する
- [x] `marked.parse()` でMarkdown→HTMLに変換する
- [x] カテゴリ（`##` 見出し）ごとにHTMLを分割し、対応するアコーディオン本文エリアに注入する

### 年度選択イベント
- [x] `<select>` の `change` イベントで以下を連動させる
  - 車体画像を差し替える（`src` 更新）
  - レギュレーションコンテンツを再読み込みする
  - 全アコーディオンを折りたたみ状態に戻す
- [x] ページ初期表示時に現在年（または 2024年）を選択状態にして自動読み込みする

### アコーディオン開閉
- [x] 各アコーディオンヘッダーのクリックで本文エリアを展開／折りたたみする
- [x] 開閉状態を `aria-expanded` 属性で管理する
- [x] スムーズなアニメーション（style.css のトランジションと連携）

## 完了条件

- 年度を変更すると画像とレギュレーション内容が切り替わる
- アコーディオンをクリックするとスムーズに開閉する
- 年度変更後は全アコーディオンが折りたたみ状態に戻る
- コンソールエラーが出ない
