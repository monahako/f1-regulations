# TICKET-005: 比較表示ロジック（compare.js）

## 概要

2年度比較画面（画面B）の動作ロジックを実装する。年度選択・時系列自動整列・車体画像の2枚表示・レギュレーションコンテンツの左右表示・diffs.json の差分表示を制御する。

## 対象ファイル

- `compare.js`（新規作成）

## 依存チケット

- TICKET-003（`getCarImage()` を再利用する）
- TICKET-004（compare.html の DOM 構造が必要）

## タスク

### 年度整列ロジック
- [x] 「比較する」ボタン押下時に Year A / Year B の値を取得する
- [x] 年A > 年B の場合（逆順）は自動で swap し、常に `小さい年 → 大きい年` に整列する
- [x] 時系列注記 `#timeline-note` を表示し「※時系列順（XXXX年 → YYYY年）で表示しています」と書き込む
- [x] 整列後の年を画面上のラベルに反映する（`#label-a`・`#label-b`）

### 車体画像表示
- [x] `getCarImage(year)` を使って整列後の年A・年Bそれぞれの画像ファイル名を取得する
- [x] `#car-image-a`・`#car-image-b` の `src` を更新する

### レギュレーションコンテンツ表示
- [x] 整列後の年Aの `.md` ファイルを `fetch` で取得し、`marked.parse()` でHTMLに変換する
- [x] 整列後の年Bの `.md` ファイルを同様に取得・変換する
- [x] 8カテゴリの本文をそれぞれ `#col-{category}-a`・`#col-{category}-b` に注入する
- [x] 年A・年Bの `.md` は並行取得（`Promise.all`）する

### diffs.json 差分表示
- [x] `diffs/diffs.json` を `fetch` で取得する（ページ初回ロード時に一度だけ取得）
- [x] キー `"{小さい年}_{大きい年}"` で差分データを参照する
- [x] 8カテゴリ分、以下を `#diff-summary-{cat}` / `#diff-details-{cat}` に注入する
  - `summary`：概要文テキスト
  - `details`：箇条書き `<li>` を生成して `<ul>` に追加する

### 表示制御
- [x] 比較ボタン押下前はコンテンツエリアを非表示にする
- [x] 比較ボタン押下後にコンテンツエリアを表示する
- [x] 再比較（別年度を選び直してボタンを押す）時に前回のコンテンツをクリアしてから再注入する

## 完了条件

- 任意の2年を選んで比較ボタンを押すと、画像・コンテンツ・差分が正しく表示される
- 逆順（例：2026→2020）で選んでも自動修正され「2020→2026」順で表示される
- 時系列注記が正しい年号で表示される
- コンソールエラーが出ない
