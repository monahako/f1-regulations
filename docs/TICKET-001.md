# TICKET-001: 共通スタイル実装（style.css）

## 概要

全ページ共通のCSS（F1公式デザイン）を実装する。カラー変数・フォント・ヘッダー・ドロップダウン・ボタンなど、両画面で使い回す基盤スタイルをここにまとめる。

## 対象ファイル

- `style.css`（新規作成）

## タスク

### 基盤設定
- [x] CSS変数（カスタムプロパティ）でデザイントークンを定義する
  - `--color-primary: #E8002D`
  - `--color-bg: #15151E`
  - `--color-text: #FFFFFF`
  - `--color-text-sub: #B0B0B0`
- [x] Google Fonts から Titillium Web を読み込む（`@import` または `<link>`）
- [x] `body` の背景色・フォント・文字色をデフォルト設定する

### ヘッダー
- [x] ヘッダー全体のスタイル（背景：プライマリレッド、高さ、padding）
- [x] ヘッダー内タイトル（サイト名）のスタイル
- [x] ヘッダー内ドロップダウン（年度選択）のスタイル

### ドロップダウン共通
- [x] `<select>` 要素のスタイル（背景・文字色・枠線・フォントサイズ）

### ボタン
- [x] 比較ボタン（「比較する」）のスタイル（背景：プライマリレッド、ホバー効果）

### アコーディオン
- [x] アコーディオンヘッダー（カテゴリ行）のスタイル（背景・文字色・カーソル・矢印アイコン）
- [x] アコーディオン本文エリアのスタイル（初期表示：非表示）
- [x] 開閉アニメーション（`max-height` トランジション等）

### 車体画像エリア
- [x] 単年表示用：画面幅65%・中央揃えのスタイル
- [x] 比較表示用：左右2枚それぞれ45%・横並びのスタイル

### 比較画面の2カラムレイアウト
- [x] カテゴリ内の左右2カラム（年A・年B）レイアウトスタイル
- [x] 差分サマリー・詳細リストのスタイル（時系列注記を含む）

## 完了条件

- 両画面（index.html・compare.html）で読み込んでもレイアウト崩れが起きない
- F1公式カラー（`#E8002D` / `#15151E`）が正しく適用されている
- Titillium Web フォントが反映されている
