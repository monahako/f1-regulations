# CLAUDE.md

このファイルは、Claude Code (claude.ai/code) がこのリポジトリで作業する際のガイダンスを提供します。

## プロジェクト概要

F1レギュレーション変遷サイト — 2000年〜2026年のF1レギュレーションを閲覧・比較できる静的日本語サイト。コンテンツはすべて準備済み。フロントエンド（HTML/CSS/JS）はこれから実装する。

デプロイ先：Vercel または Netlify（GitHub連携、自動デプロイ）。ビルド工程なし。

## 技術スタック

- バニラ HTML5 / CSS / JavaScript（フレームワーク不使用）
- `marked.js`（CDN経由）でクライアントサイドのMarkdown → HTML変換
- 完全静的サイト。サーバー・ビルドプロセス不要

## データ資産

| 資産 | 場所 | 説明 |
|---|---|---|
| レギュレーション本文 | `regulations/YYYY.md` | 年ごとに1ファイル（2000〜2026）、各8カテゴリ |
| 年間差分データ | `diffs/diffs.json` | 351通りの比較。キーは `"YYYY_YYYY"`（小さい年を先） |
| カーイメージ | `images/CarImage_*.png` | 時代別5枚（下記マッピング参照） |

**イメージの時代マッピング：**
- `CarImage_2000-2008.png` → 2000〜2008年
- `CarImage_2009-2011.png` → 2009〜2011年
- `CarImage_2012-2021.png` → 2012〜2021年
- `CarImage_2022-2025.png` → 2022〜2025年
- `CarImage_2026.png` → 2026年

**diffs.json の構造（1エントリ例）：**
```json
"2020_2026": {
  "engine": { "summary": "...", "details": ["...", "..."] },
  "chassis": { ... },
  ...カテゴリ計8個
}
```

## 画面構成

**`index.html` — 単年表示：**
- 年選択ドロップダウン（2000〜2026）
- カーイメージを中央に表示（幅65%）
- 8カテゴリのアコーディオン（初期状態：すべて折りたたみ）、内容は `regulations/YYYY.md` から取得

**`compare.html` — 年比較表示：**
- 年A・年Bの2つのドロップダウン
- カーイメージを左右並列表示（各45%）
- 常に時系列順（古い年 → 新しい年）で表示。逆順選択は自動修正
- 表示注記：`※時系列順（XXXX年 → YYYY年）で表示しています`
- `diffs.json` の差分を左右2カラムで表示

## 8カテゴリ（固定順序・全年共通）

1. ① エンジン／パワーユニット
2. ② 車体／シャシー
3. ③ 空力・タイヤ・ホイール
4. ④ レースルール（Format）
5. ⑤ 安全・健康
6. ⑥ コスト・開発制限
7. ⑦ メディア・運営・公平性
8. ⑧ 環境・サステナビリティ

## 重要な制約

- `diffs.json` のキーは必ず `"小さい年_大きい年"` 形式（例：`"2020_2026"`。`"2026_2020"` は不可）。
- 比較画面は、どちらのドロップダウンで選んでも常に時系列順に自動整列する。
- `.md` ファイルと `diffs.json` のカテゴリ見出し・キーは全27年で完全一致させる。
- サイト言語は日本語のみ（多言語対応は将来フェーズ）。
